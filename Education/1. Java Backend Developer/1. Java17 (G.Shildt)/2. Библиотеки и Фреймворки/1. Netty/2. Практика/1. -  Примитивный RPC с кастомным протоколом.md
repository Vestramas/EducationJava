# Суть проекта:
Клиент работает в консоли. Пишет команду например: `ECHO Hello world`, запрос содержащий команду и payload формируются в простой пакет протокола, и этот пакет затем сериализуется в байты.

Сервер принимает байты, десериализует их в пакет протокола, и парсит оттуда данные (достает команду и payload). В зависимости от вызываемой команды вызывается соответствующий обработчик команды.

Обработчик команды формирует результат команды (если команда была `ECHO Hello world`, то результатом будет строка `Hello world`)
Полученная строка помещается в пакет протокола, и затем пакет сериализуется в байты.

Клиент получает байты, десериализует, и выводит результат выполнения команды

Доступные команды:
`ECHO` - вывод текста
`TIME` - получение текущего времени сервера
`UPPER` - вывод текста ПРОПИСНЫМИ БУКВАМИ
`HELP` - вывод списка доступных команд

`LOGIN/LOGOUT` - команды для входа и выхода (не сделал, поскольку интереснее будет перейти на другой учебный пример)

## Архитектура
### Архитектура сервера
![[Pasted image 20260615152042.png]]
Пришедшее сообщение, декодируется декодером, и отправляется в обработчик. В обработчике вызывается определенный класс для выполнения команды.
Используется паттерн Стратегия, для унифицированного API методов.

После выполнения команды, результат направляется на OUT направления pipeline, где находится кодировщик данных.


### Архитектура клиента
![[Pasted image 20260615154426.png]]
Есть обработчики на IN и OUT направлении:
- IN - чтение ответа от сервера
- OUT - логирование при подключении и отключении соединения

В конце Pipeline будут кодировщики. Именно они будут преобразовывать наши данные в байты, когда будет вызываться `RpcClient.send(Command, Payload)`.

Все управление будет через консоль. Поэтому понадобится класс `ConsoleManager`, который будет читать команды с консоли, и вызывать методы `RpcClient`

```java
import java.util.HashMap;  
import java.util.Map;  
  
public enum Command {  
    LOGIN((byte) 1),  
    LOGOUT((byte) 2),  
    ECHO((byte) 3),  
    TIME((byte) 4),  
    UPPER((byte) 5),  
    HELP((byte) 6),;  
  
    private final byte code;  
    private static final Map<Byte, Command> BY_CODE = new HashMap<>();  
  
    static {  
        BY_CODE.put(LOGIN.code, LOGIN);  
        BY_CODE.put(LOGOUT.code, LOGOUT);  
        BY_CODE.put(ECHO.code, ECHO);  
        BY_CODE.put(TIME.code, TIME);  
        BY_CODE.put(UPPER.code, UPPER);  
        BY_CODE.put(HELP.code, HELP);  
    }  
  
    Command(byte code) {  
        this.code = code;  
    }  
    public static Command getCommand(byte code) {  
        Command command = BY_CODE.get(code);  
        if (command == null) {  
            throw new IllegalArgumentException("Unknown command code: " + code);  
        }  
        return command;  
    }  
  
    public byte getCode() {  
        return code;  
    }  
}
```

```java
package remote_command_protocol.protocol;  
  
import lombok.Getter;  
  
import java.util.Arrays;  
  
@Getter  
public class Protocol {  
    public static final int HEADER_SIZE = 12;  
    public static final short MAGIC = 0xCA;  
    private final byte VERSION;  
    private final Command COMMAND;  
    private final int REQUEST_ID;  
    private final int BODY_LENGTH;  
  
    private final byte[] BODY;  
  
    public Protocol(byte VERSION, Command COMMAND, int REQUEST_ID, byte[] body) {  
        this.VERSION = VERSION;  
        this.COMMAND = COMMAND;  
        this.REQUEST_ID = REQUEST_ID;  
        this.BODY_LENGTH = body.length;  
        this.BODY = body;  
    }  
  
    @Override  
    public String toString() {  
        return "Protocol{" +  
            "VERSION=" + VERSION +  
            ", COMMAND=" + COMMAND +  
            ", REQUEST_ID=" + REQUEST_ID +  
            ", BODY_LENGTH=" + BODY_LENGTH +  
            ", BODY=" + Arrays.toString(BODY) +  
            '}';  
    }  
}
```

## Декодер
```java
package remote_command_protocol.codec;  
  
import io.netty.buffer.ByteBuf;  
import io.netty.buffer.ByteBufUtil;  
import io.netty.channel.ChannelHandler;  
import io.netty.channel.ChannelHandlerContext;  
import io.netty.handler.codec.ByteToMessageDecoder;  
import remote_command_protocol.protocol.Command;  
import remote_command_protocol.protocol.Protocol;  
  
import java.util.List;  
import java.util.logging.Logger;  
  
public class ProtocolDecoder extends ByteToMessageDecoder {  
    private static final Logger log = Logger.getLogger(ProtocolDecoder.class.getName());  
  
    @Override  
    protected void decode(
	    ChannelHandlerContext ctx, 
	    ByteBuf in, 
	    List<Object> out
	) throws Exception {  
        log.info(
	        "Receive hex dump:\n" + ByteBufUtil.hexDump(in));  
  
		// Проверяем, что размер пришедших данныъ НЕ МЕНЬШЕ чем 
		// заголовок протокола. 
		// Так как если данных меньше, то мы не сможем распарсить 
		// даже заголовок, который содержит необходимые данные 
		// для декодирования
        if(in.readableBytes() < Protocol.HEADER_SIZE){  
            log.warning(  
                "The size of the received bytes is less than the protocol header. Received bytes:" + in.readableBytes());  
            return;  
        }  
		
		// Устанавливаем "закладку" в байтах, чтобы в случае чего вернуться сюда, и дождаться остальных данных в payload  
        in.markReaderIndex(); 
        
 
        short magic = in.readShort();  
        if (magic != Protocol.MAGIC) {  
            log.warning(
				"This package does not belong to the binary protocol. An invalid preamble has been received: " 
				+ Integer.toHexString(magic)
			);  
            in.resetReaderIndex();  
            return;  
        }  
  
        log.info("Parsing header");  
        byte version = in.readByte();  
        Command command = Command.getCommand(in.readByte());  
        int requestId = in.readInt();  
        int length = in.readInt();  
  
        if (in.readableBytes() < length){  
            in.resetReaderIndex(); // Возвращаемся на "закладку" 
            log.warning("The message payload was not received in full. Received bytes:" + in.readableBytes());  
            return;  
        }  
  
        log.info("Parsing data");  
        byte[] body = new byte[length];  
        in.readBytes(body);  
  
        Protocol protocol = new Protocol(  
            version,  
            command,  
            requestId,  
            body  
        );  
  
        log.info("Decoded message:\n" + protocol.toString());  
  
        out.add(protocol);  
    }  
}
```

## Кодер
```java
package remote_command_protocol.codec;  
  
import io.netty.buffer.ByteBuf;  
import io.netty.buffer.ByteBufUtil;  
import io.netty.channel.ChannelHandler;  
import io.netty.channel.ChannelHandlerContext;  
import io.netty.handler.codec.MessageToByteEncoder;  
import remote_command_protocol.protocol.Protocol;  
  
import java.util.logging.Logger;  
  
public class ProtocolEncoder extends MessageToByteEncoder<Protocol> {  
    private static final Logger log = Logger.getLogger(ProtocolEncoder.class.getName());  
  
    @Override  
    protected void encode(ChannelHandlerContext ctx, Protocol msg, ByteBuf out) throws Exception {  
        log.info("Encoded message:\n" + msg.toString());  
  
        out.writeShort(Protocol.MAGIC)  
            .writeByte(msg.getVERSION())  
            .writeByte(msg.getCOMMAND().getCode())  
            .writeInt(msg.getREQUEST_ID())  
            .writeInt(msg.getBODY_LENGTH())  
            .writeBytes(msg.getBODY());  
  
        log.info("Encoded hex dump:\n" + ByteBufUtil.hexDump(out));  
    }  
}
```

### Обработчик канала
Теперь опишем главный и единственный обработчик каналов, который будет вызываться после того, как декодер декодирует байты в сообщение протокола

> [!WARNING] ИСПОЛЬЗОВАНИЕ ПАТТЕРНА "СТРАТЕГИЯ"
> Чтобы использовать паттерн стратегия, можно использовать Map, и по ключу, доставать необходимые класс реализацию

Определенный обработчик команды будем вызывать на основе команды, которая пришла по сети. Для этого создадим Map, с ключ-значение: Команда-Обработчик

```java
package remote_command_protocol.server.handler;  
  
import io.netty.channel.ChannelHandlerContext;  
import io.netty.channel.SimpleChannelInboundHandler;  
import remote_command_protocol.protocol.Command;  
import remote_command_protocol.protocol.Protocol;  
import remote_command_protocol.server.executor.RpcExecutor;  
import remote_command_protocol.server.executor.impl.EchoExecutor;  
import remote_command_protocol.server.executor.impl.HelpExecutor;  
import remote_command_protocol.server.executor.impl.TimeExecutor;  
import remote_command_protocol.server.executor.impl.UpperExecutor;  
  
import java.nio.charset.StandardCharsets;  
import java.util.HashMap;  
import java.util.Map;  
import java.util.logging.Logger;  
  
public class RpcServerInboundHandler 
extends SimpleChannelInboundHandler<Protocol> 
{  
    private static final Logger log = 
	    Logger.getLogger(
		    RpcServerInboundHandler.class.getName()
		);  
	
	// Map с обработчиками
    private static Map<Command, RpcExecutor> executors = 
	    new HashMap<Command, RpcExecutor>();  
  
    static {  
        executors.put(Command.ECHO, new EchoExecutor());  
        executors.put(Command.TIME, new TimeExecutor());  
        executors.put(Command.UPPER, new UpperExecutor());  
        executors.put(Command.HELP, new HelpExecutor());  
    }  
  
    @Override  
    protected void channelRead0(ChannelHandlerContext ctx, Protocol msg) throws Exception {  
        RpcExecutor rpcExecutor = executors.get(msg.getCOMMAND());  
  
        if(rpcExecutor == null) {  
            log.warning("Unknown command: " + msg.getCOMMAND());  
            return;  
        }  
  
        log.info("RPC executor call");  
        String result = rpcExecutor.execute(msg);  
  
        log.info("Preparing responce");  
        Protocol response =  
            new Protocol(  
                msg.getVERSION(),  
                Command.ECHO,  
                msg.getREQUEST_ID(),  
                result.getBytes(StandardCharsets.UTF_8)  
            );  
  
        ctx.writeAndFlush(response)  
           .addListener(f -> {  
               if (f.isSuccess()) {  
                   log.info("Response sent successfully");  
               }  
           });  
    }  
  
    @Override  
    public void exceptionCaught(ChannelHandlerContext ctx, Throwable cause) throws Exception {  
        log.warning("Exception on handle message: " + cause.getMessage());  
        ctx.close();  
    }  
}
```

```java
package remote_command_protocol.server;  
  
import io.netty.bootstrap.ServerBootstrap;  
import io.netty.channel.ChannelFuture;  
import io.netty.channel.ChannelInitializer;  
import io.netty.channel.EventLoopGroup;  
import io.netty.channel.nio.NioEventLoopGroup;  
import io.netty.channel.socket.SocketChannel;  
import io.netty.channel.socket.nio.NioServerSocketChannel;  
import io.netty.handler.logging.LogLevel;  
import io.netty.handler.logging.LoggingHandler;  
import remote_command_protocol.server.handler.RpcServerPipelineInitializer;  
  
import java.net.InetSocketAddress;  
import java.util.logging.Logger;  
  
public class RpcServer {  
  
    private static final Logger log = Logger.getLogger(RpcServer.class.getName());  
    private final int port;  
  
    public RpcServer(int port) {  
        this.port = port;  
    }  
  
    public static void main(String[] args) {  
        new RpcServer(8080).start();  
    }  
  
    public void start() {  
        EventLoopGroup bossGroup = new NioEventLoopGroup(1);  
        EventLoopGroup workerGroup = new NioEventLoopGroup();  
  
        try{  
            ServerBootstrap bootstrap = new ServerBootstrap();  
            bootstrap.group(bossGroup, workerGroup)  
                     .channel(NioServerSocketChannel.class)  
                     .localAddress(new InetSocketAddress(port))  
                     .handler(new LoggingHandler(LogLevel.INFO))  
                     .childHandler(
	                     new RpcServerPipelineInitializer());  
  
  
            ChannelFuture future = bootstrap.bind().sync();  
            log.info("Server started on port " + port);  
  
            future.channel().closeFuture().sync();  
  
        } catch (Exception e){  
            e.printStackTrace();  
        }  
    }  
}
```

## 4.2 Консольный менеджер и Клиент
### Бутстрапинг клиента
```java
package remote_command_protocol.client.application;  
  
import io.netty.bootstrap.Bootstrap;  
import io.netty.channel.Channel;  
import io.netty.channel.ChannelFuture;  
import io.netty.channel.EventLoopGroup;  
import io.netty.channel.nio.NioEventLoopGroup;  
import io.netty.channel.socket.nio.NioSocketChannel;  
import remote_command_protocol.client.handler.RpcClientPipelineInitializer;  
import remote_command_protocol.protocol.Command;  
import remote_command_protocol.protocol.Protocol;  
  
import java.net.InetSocketAddress;  
import java.nio.charset.StandardCharsets;  
import java.util.concurrent.atomic.AtomicInteger;  
import java.util.logging.Logger;  
  
public class RpcClient {  
    private static final Logger log = 
	    Logger.getLogger(RpcClient.class.getName());  
    
    private final AtomicInteger requestIdGenerator = 
	    new AtomicInteger();  
	    
    private Channel channel;  
    private EventLoopGroup group;  
    private final int port = 8080;  
  
    public void start() {  
        group = new NioEventLoopGroup(1);  
  
        Bootstrap bootstrap = new Bootstrap();  
        bootstrap  
            .group(group)  
            .channel(NioSocketChannel.class)  
            .remoteAddress(new InetSocketAddress(8080))  
            .handler(new RpcClientPipelineInitializer());  
  
        ChannelFuture future = 
	        bootstrap.connect().syncUninterruptibly(); 
	         
        channel = future.channel();  
    }  
  
    public void close(){  
        channel.disconnect();  
        channel.close();  
        group.shutdownGracefully();  
    }  
  
    public void send(Command command, String payload) {  
        Protocol request = new Protocol(  
            (byte) 1,  
            command,  
            requestIdGenerator.incrementAndGet(),  
            payload.getBytes(StandardCharsets.UTF_8));  
  
        channel.writeAndFlush(request)  
            .addListener(future -> {  
                if (future.isSuccess()) {  
                    log.info("Message sent successfully");  
                }  
            });  
    }  
}
```

```java
	    new Scanner(System.in);  
  
    public static void main(String[] args) {  
        RpcClient rpcClient = new RpcClient();  
        rpcClient.start();  
  
        System.out.println(
	        "Welcome to the remote command protocol client");  
        System.out.println(
	        "To see available command - <HELP>\nTo disconnect - <exit>");  
  
        String action;  
        while (true) {  
            System.out.print(">> ");  
            action = scanner.nextLine();  
  
            if(action.equals("exit")) {  
                rpcClient.close();  
                return;  
            }  
  
            ConsoleData consoleData = parseConsoleInput(action);  
            if(consoleData == null) {  
                continue;  
            }  
  
            rpcClient.send(
	            consoleData.command(), consoleData.payload());  
        }  
    }  
  
    public static ConsoleData parseConsoleInput(String input) {  
        String[] tokens = input.split(" ");  
  
        String payload = Arrays.stream(tokens)  
            .skip(1)  
            .collect(Collectors.joining(" "));  
  
        try {  
            Command command = Command.valueOf(tokens[0]);  
  
            return new ConsoleData(command, payload);  
        } catch (IllegalArgumentException e){  
            log.warning("Illegal command: " + tokens[0]);  
            return null;  
        }  
    }  
}
```

В консоль будет вводиться строка вида: `<КОМАНДА> <PAYLOAD>`, и эту строку нужно парсить.

А чтобы удобно доставать информацию из спрашенной строки, создадим класс обертку для этих данных - `record`
```java
package remote_command_protocol.client.application;  
  
import remote_command_protocol.protocol.Command;  
  
public record ConsoleData(Command command, String payload){}
```
