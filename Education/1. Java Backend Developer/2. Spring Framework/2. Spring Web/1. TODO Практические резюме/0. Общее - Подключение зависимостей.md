# 1. Подключение зависимостей

##### Минимальная зависимости для Spring MVC 
==1. Spring Context==
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-context</artifactId>
    <version>5.3.30</version> <!-- или актуальная версия -->
</dependency>
```
Включает:
- **Spring Core** (IoC контейнер, управление бинами)
- **Spring Beans** (механизм работы с бинами)
- **Spring Context** (расширение контейнера, поддержка аннотаций `@Component`, `@Autowired` и других)
***
==2. Spring WebMVC== - сердце Spring MVC
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-webmvc</artifactId>
    <version>5.3.30</version>
</dependency>
```
Этот модуль **расширяет** `spring-web`, добавляя реализацию **MVC (Model-View-Controller)**:

- **`DispatcherServlet`** – главный сервлет Spring MVC, маршрутизирующий HTTP-запросы
- **Аннотации контроллеров**: `@Controller`, `@RestController`, `@RequestMapping`
- **Механизм обработки представлений** (JSP, Thymeleaf, Freemarker)
- **Формирование моделей и передача данных в представления (`ModelAndView`)**
- **Поддержка валидации (`@Valid`, `BindingResult`)**
***
==3. Servlet API== (`jakarta.servlet-api` или `javax.servlet-api`)
```xml
<dependency>
    <groupId>jakarta.servlet</groupId>
    <artifactId>jakarta.servlet-api</artifactId>
    <version>5.0.0</version>
    <scope>provided</scope>
</dependency>
```
Нужно выбрать одно из пространств имен jakatra или javax, в зависимости от версии контейнера сервлетов (сервер Tomtac и др.)

**Включает:**
- Определение стандартных интерфейсов сервлетов (`HttpServletRequest`, `HttpServletResponse`, `Filter` и т. д.).
- Механизм обработки HTTP-запросов и ответов.
- Spring MVC использует сервлеты, такие как `DispatcherServlet`

***
#### Дополнительные зависимости
==1. Spring-Web==
```xml
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-web</artifactId>
    <version>5.3.30</version>
</dependency>
```
Этот модуль является базовым для работы с веб-приложениями в Spring и включает:
- **Поддержку работы с HTTP**
    - Клиенты HTTP (`RestTemplate`, `WebClient`)
    - Поддержку `HttpMessageConverter` (конвертация запросов и ответов в XML, JSON и т. д.)
- **Поддержку работы с CORS** (Cross-Origin Resource Sharing)
- **Работу с Multipart (загрузка файлов)**
- **Базовые фильтры и интерсепторы**
***
==2. **Jackson== (`jackson-databind`)**
```xml
<dependency>
    <groupId>com.fasterxml.jackson.core</groupId>
    <artifactId>jackson-databind</artifactId>
    <version>2.16.0</version>
</dependency>
```
**Включает:**
- Механизм сериализации/десериализации JSON в объекты Java (`@JsonProperty`, `@JsonIgnore`).
- Автоматическую поддержку `@ResponseBody` в Spring MVC.
***
==3. **Logback== (`logback-classic`)**
```xml
<dependency>
    <groupId>ch.qos.logback</groupId>
    <artifactId>logback-classic</artifactId>
    <version>1.4.14</version>
</dependency>
```
**Включает:**
- Гибкую систему логирования для Spring-приложений.
- Поддержку разных уровней логов (`INFO`, `DEBUG`, `ERROR`).
- Возможность настройки логирования через `logback.xml`.
***
==4 **JSTL== (JavaServer Pages Standard Tag Library)**
```xml
<dependency>
    <groupId>javax.servlet</groupId>
    <artifactId>jstl</artifactId>
    <version>1.2</version>
</dependency>
```
**Включает:**
- Набор стандартных JSP-тегов (`<c:forEach>`, `<c:if>`, `<fmt:formatNumber>`).
- Упрощенную работу с представлениями.
***
==5.**Thymeleaf**== - шаблонизатор представлений
#### Если используется Spring 5
1. Движок Thymleaf
```xml
<dependency>
    <groupId>org.thymeleaf</groupId>
    <artifactId>thymeleaf</artifactId>
    <version>3.1.2.RELEASE</version>
</dependency>
```
Включает:
- **Базовый движок Thymeleaf**
- **Поддержку шаблонов HTML/XML**
- **Атрибуты `th:` (`th:text`, `th:each`, `th:if` и т. д.)**
- **Механизм обработки фрагментов (`th:replace`, `th:insert`)**

2. Интеграция Thymleaf и Spring
```xml
<dependency>
    <groupId>org.thymeleaf</groupId>
    <artifactId>thymeleaf-spring5</artifactId>
    <version>3.1.2.RELEASE</version>
</dependency>
```
**Включает**
- **Интеграцию Thymeleaf с Spring 5 и Spring MVC**
- **Поддержку модели `Model` из контроллеров Spring (`model.addAttribute(...)`)**
- **Автоматическое связывание данных форм (`@ModelAttribute`)**
- **Работу с `@Controller` и `@RestController`**
#### Если используется Spring 6
```xml
<dependency>
    <groupId>org.thymeleaf</groupId>
    <artifactId>thymeleaf-spring6</artifactId>
    <version>3.1.2.RELEASE</version>
</dependency>
```
Содержит:
✅Включает **базовый движок Thymeleaf** (то же самое, что `thymeleaf`)  
✅ Добавляет **интеграцию с Spring 6 и Spring MVC** (то же самое, что `thymeleaf-spring5`, но для Spring 6)  
✅ Позволяет передавать данные из **контроллеров Spring MVC** (`Model`, `@ModelAttribute`)  
✅ Поддерживает работу с **формами и валидацией**  
✅ Совместима с **Spring Boot 3**
