> [!NOTE] ЗАМЕТКИ
> Классов конфигураторов МОЖЕТ быть НЕСКОЛЬКО

==Классы==/файлы.xml ==конфигураторы==, ==можно разделить на узконаправленные классы. ==

Например, можно создать класс конфигуратор, который определяет бины для взаимодействия с БД
И отдельно создать класс конфигуратор, отвечающий за создание бинов для шаблонизаторов, контроллеров и прочего
