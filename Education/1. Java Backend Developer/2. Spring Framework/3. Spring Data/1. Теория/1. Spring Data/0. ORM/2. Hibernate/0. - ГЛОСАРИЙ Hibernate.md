# 1. Hibernate
Реализация спецификации JPA (JPA-провайдер) + собственный нативный API.

Может **использоваться как через** 
- *стандартные JPA-интерфейсы*, 
- ***так и через нативные классы***.

---
# 2. SessionFactory - аналог `EntityManagerFactory` в JPA
Главная фабрика Hibernate (**аналог `EntityManagerFactory` в JPA**). 

Потокобезопасна, создаётся один раз при запуске. 
Хранит метаданные маппингов и настройки подключения к БД.

---
# 3. Session аналог `EntityManager` в JPA
Основной объект для работы с БД в Hibernate (**аналог `EntityManager` в JPA**). 
**Не потокобезопасен.** 

Управляет персистентным контекстом, выполняет CRUD-операции и запросы.

---
# 4. Contextual Sessions
Механизм Hibernate, при котором сам Hibernate управляет одной текущей Session на транзакцию. 

Включается через `hibernate.current_session_context_class`.

---
# 5. LocalSessionFactoryBean

Spring-класс (`FactoryBean`) **для создания `SessionFactory` в Spring IoC-контейнере.** 

Принимает `DataSource`, маппинги и свойства Hibernate.

---
# 6. LocalSessionFactoryBuilder
**Удобный builder для программного создания** `SessionFactory` без `FactoryBean`. 

Интегрируется с `@Bean`-конфигурацией.

---
# 7. HibernateTransactionManager
==Реализация== `PlatformTransactionManager` **для управления транзакциями через нативный Hibernate.** 

Привязывает `Session` к текущему потоку через `ThreadLocal`.

---
# 8. HibernateJpaVendorAdapter
**Адаптер, позволяющий использовать Hibernate как JPA-провайдер** с автоматической настройкой специфичных для Hibernate параметров.

---
# 9. hibernate.cfg.xml
==Нативный конфигурационный файл Hibernate== (аналог `persistence.xml` для JPA). 

Может использоваться вместо `persistence.xml` при работе без JPA.

---
# 10. HibernateException
==Базовое unchecked-исключение Hibernate.==

Все исключения фреймворка наследуются от него. 

Spring может преобразовывать их в иерархию `DataAccessException`.

---
# 11. Background Bootstrapping
Возможность инициализировать `SessionFactory` в фоновом потоке параллельно с загрузкой приложения. Ускоряет старт приложения.
