Spring **предоставляет безопасные и разумные ДЕФОЛТЫ для инфраструктуры**, и позволяет точечно переопределить любое поведение при необходимости.

Поэтому зачастую можно использовать следующие настройки:
```java
http.oauth2Login(Customizer.withDefaults());
```

То есть, используй `Customizer.withDefaults()`, если не нужна особая настройка для приложения
