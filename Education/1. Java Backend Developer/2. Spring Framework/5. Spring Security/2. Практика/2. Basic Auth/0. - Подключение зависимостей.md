Создадим простое **REST приложение, которое будет сохранять пользователей в БД.**

На основе этого, затем добавим Spring Security

---

Чтобы сохранять данные пользователя в БД, нужно определиться, как мы это будем делать

Обговорим что нам будет нужно:
- Для простоты, будет использовать спецификацию `JPA` вместе с `Hibernate`. 
	
- В качестве БД будем использовать `PostgreSQL`, поэтому нужен драйвер для него
	
- Чтобы возвращать пользователю информацию о пользователях в БД в виде JSON, нужно подключить `Jackson` (с модулем для времени)
	
- Чтобы мапить DTO в Model, и наоборот, воспользуемся библиотекой `Mapstruct`
	
- Чтобы вручную не писать конструкторы, get/set методы, подключим `Lombok`
	
- Hibernate использует `SLF4j` для логирования, поэтому и его тоже подключим

В родительском pom.xml в dependencyManagmet добавим зависимости с их версиями

```xml
<groupId>org.example</groupId>
  <artifactId>SecurityExamples</artifactId>
  <packaging>pom</packaging>

  <modules>
    <module>basic-auth</module>
    <module>in-memory-auth</module>
  </modules>

  <version>1.0-SNAPSHOT</version>

  <name>SecurityExamples Maven Webapp</name>
  <url>http://maven.apache.org</url>

  <properties>
    <jakarta-servlet-version>6.0.0</jakarta-servlet-version>
    <lombok-version>1.18.42</lombok-version>
    <postgres-driver-version>42.6.0</postgres-driver-version>
    <slf4j-version>2.0.17</slf4j-version>
    <mapstruct-version>1.6.3</mapstruct-version>
    <jackson-version>2.20.1</jackson-version>

    <spring-version>6.2.5</spring-version>
    <spring-jpa-version>3.4.4</spring-jpa-version>
    <hibernate-version>6.5.2.Final</hibernate-version>
  </properties>

  <dependencyManagement>
    <dependencies>
      <!-- Lombok -->
      <dependency>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok</artifactId>
        <version>${lombok-version}</version>
        <scope>compile</scope>
      </dependency>

      <!-- Mapstruct -->
      <dependency>
        <groupId>org.mapstruct</groupId>
        <artifactId>mapstruct</artifactId>
        <version>${mapstruct-version}</version>
        <scope>compile</scope>
      </dependency>

      <dependency>
        <groupId>org.mapstruct</groupId>
        <artifactId>mapstruct-processor</artifactId>
        <version>${mapstruct-version}</version>
        <scope>compile</scope>
      </dependency>

      <!-- Jackson-core -->
      <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-core</artifactId>
        <version>${jackson-version}</version>
        <scope>compile</scope>
      </dependency>

      <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-databind</artifactId>
        <version>${jackson-version}</version>
        <scope>compile</scope>
      </dependency>

      <dependency>
        <groupId>com.fasterxml.jackson.datatype</groupId>
        <artifactId>jackson-datatype-jsr310</artifactId>
        <version>${jackson-version}</version>
      </dependency>

      <!-- Spring Data JPA -->
      <dependency>
        <groupId>org.springframework.data</groupId>
        <artifactId>spring-data-jpa</artifactId>
        <version>${spring-jpa-version}</version>
        <scope>compile</scope>
      </dependency>

      <dependency>
        <groupId>org.springframework</groupId>
        <artifactId>spring-orm</artifactId>
        <version>${spring-version}</version>
      </dependency>

      <!-- Hibernate -->
      <dependency>
        <groupId>org.hibernate.orm</groupId>
        <artifactId>hibernate-core</artifactId>
        <version>${hibernate-version}</version>
        <scope>compile</scope>
      </dependency>

      <!-- PostgreSQL -->
      <dependency>
        <groupId>org.postgresql</groupId>
        <artifactId>postgresql</artifactId>
        <version>${postgres-driver-version}</version>
        <scope>compile</scope>
      </dependency>

      <!-- Slf4j-api -->
      <dependency>
        <groupId>org.slf4j</groupId>
        <artifactId>slf4j-api</artifactId>
        <version>${slf4j-version}</version>
        <scope>compile</scope>
      </dependency>

      <!-- Slf4j-simple-provider -->
      <dependency>
        <groupId>org.slf4j</groupId>
        <artifactId>slf4j-simple</artifactId>
        <version>${slf4j-version}</version>
        <scope>compile</scope>
      </dependency>

    </dependencies>
  </dependencyManagement>
```
