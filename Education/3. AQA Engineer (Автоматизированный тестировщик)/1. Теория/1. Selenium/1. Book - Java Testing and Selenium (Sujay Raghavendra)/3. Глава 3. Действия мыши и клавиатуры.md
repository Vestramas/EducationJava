В этой главе вы погрузитесь в концепцию **цепочек действий (action chains)** — мощного инструмента для имитации сложных взаимодействий пользователя с веб-приложениями. 

**Цепочки действий позволяют автоматизировать множество шагов, объединяя одновременный ввод с клавиатуры и мыши**. 

Этот функционал необходим для тщательного тестирования веб-приложений. 
С их помощью вы можете автоматизировать различные действия клавиатуры (нажатия клавиш, ввод текста), а также действия мыши (клики, перетаскивание) и прокрутку. 
Эта глава познакомит вас с основами создания и выполнения цепочек действий.

---
## Цепочки действий (Action Chains)

Термин "цепочка действий" означает набор инструкций, выполняемых последовательно для имитации действий пользователя в веб-приложении с целью тестирования его функциональности.

---
# 1. Действия мыши (Mouse Actions)

Мышь выполняет различные операции: клики, перетаскивание, перемещение и т.д. Эти операции используются для имитации движений пользователя и тестирования функций приложения в браузере.

---
## Клик (Click)

Метод `click()` — один из самых часто используемых. *Он перемещает указатель в центр элемента и выполняет клик левой кнопкой мыши.*
```java
import java.util.concurrent.TimeUnit;
import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;

public class Mouse {
    public static void main(String[] args) throws Exception {
        WebDriver driver = new ChromeDriver();
        driver.get("https://www.selenium.dev");

        // Пауза для загрузки страницы
        TimeUnit.SECONDS.sleep(5);

        // Находим ссылку "Downloads"
        WebElement downloadLink = 
	        driver.findElement(By.linkText("Downloads"));
        
        // Выполняем клик
        downloadLink.click();
    }
}
```

> **Примечание:** Начиная с ChromeDriver версии 114, путь к драйверу можно не указывать.

---
## Двойной клик (Double Click)

Двойной клик — это два нажатия левой кнопкой мыши. Указатель сначала перемещается в центр элемента.
```java
import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.interactions.Actions;

public class DoubleClickExample {
    public static void main(String[] args) {
        WebDriver driver = new ChromeDriver();
        driver.get("https://example.com");

        WebElement element = 
	        driver.findElement(By.id("element-id"));


        Actions actions = new Actions(driver);
        actions.doubleClick(element).perform();

        driver.quit();
    }
}
```

Этот метод может использоваться для открытия элементов, выделения текста или активации определенных элементов управления.

---
## Контекстный клик (ПКМ) — Context Click

Это действие перемещает указатель в центр элемента и выполняет клик правой кнопкой мыши.
```java
import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.interactions.Actions;

public class ContextClickExample {
    public static void main(String[] args) {
        WebDriver driver = new ChromeDriver();
        driver.get("https://example.com");

        WebElement element = 
	        driver.findElement(By.id("element-id"));

        Actions actions = new Actions(driver);
        actions.contextClick(element).perform();

        driver.quit();
    }
}
```

Обычно используется для тестирования контекстных меню.

---
## Нажать и удерживать (Click and Hold)

В этом методе указатель перемещается в центр элемента, нажимается левая кнопка мыши и **не отпускается**.
```java
import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.interactions.Actions;

public class ClickAndHoldExample {
    public static void main(String[] args) {
        WebDriver driver = new ChromeDriver();
        driver.get("https://example.com");

        WebElement element = 
	        driver.findElement(By.id("element-id"));

        Actions actions = new Actions(driver);
        actions.clickAndHold(element).perform();

        // Дополнительные действия (например, перемещение)
        actions.release().perform(); // Не забудьте отпустить!

        driver.quit();
    }
}
```

Чаще всего используется в сценариях перетаскивания (drag-and-drop).

---
## Отпускание (release())

Метод для отпускания ранее зажатой кнопки мыши. Классический пример — перетаскивание: элемент зажимается, перетаскивается в нужное место и отпускается.

```java
Actions actions = new Actions(driver);

actions.clickAndHold(someElement)  // Зажать элемент
       .moveToElement(anotherElement) // Переместить к другому элементу
       .release()                    // Отпустить кнопку
       .perform();
```

---
## Выполнение (perform())

Метод `perform()` **выполняет все накопленные действия**. Используется, когда нужно выполнить одно или несколько действий подряд.

```java
Actions actions = new Actions(driver);

actions.moveToElement(someElement)
       .click()
       .perform(); // Выполняет клик по элементу
```

---
## Пауза (pause())

**Пауза — это задержка между действиями.** Может использоваться **для имитации реалистичного поведения пользователя** или ожидания появления элемента.
```java
import java.time.Duration;

Actions actions = new Actions(driver);

actions.moveToElement(someElement)
       .click()
       .pause(Duration.ofMillis(1000)) // Пауза 1 секунда
       .click(anotherElement)
       .perform();
```

---
## Сброс (reset())

Сбрасывает или **очищает текущее состояние всех накопленных действий** в построителе действий.
```java
Actions actions = new Actions(driver);

actions.moveToElement(someElement).click().perform();

actions.reset(); // Сбрасываем построитель

// Строим новую последовательность
actions.moveToElement(anotherElement).contextClick().perform();
```

# 2. Перемещения мыши (Mouse Movements)

## Переместиться к элементу (moveToElement())

**Перемещает курсор мыши в центр указанного веб-элемента**, имитируя наведение (hover). Используется для тестирования выпадающих меню или скрытых элементов.
```java
// Локализуем элемент
WebElement element = driver.findElement(By.id("some-id"));

// Создаем объект Actions
Actions actions = new Actions(driver);

// Выполняем наведение
actions.moveToElement(element).perform();
```

---
## Перемещение по смещению (moveByOffset())

Перемещает указатель мыши **на заданное количество пикселей (смещение).** 

Отсчет начинается от **верхнего левого угла экрана** (viewport), если до этого мышь никуда не перемещалась.
```java
Actions actions = new Actions(driver);

// Переместить мышь на 50px вправо и 100px вниз
actions.moveByOffset(50, 100).perform();
```

> **Важно:**
> - Положительное значение X — движение вправо, отрицательное — влево.
>     
> - Положительное значение Y — движение вниз, отрицательное — вверх.

---
## Смещение от элемента (Offset from Element)

Сначала перемещает мышь в центр элемента, затем смещает ее на указанные значения X и Y.
```java
WebElement targetElement = 
	driver.findElement(By.id("someElementId"));

Actions actions = new Actions(driver);

// Сначала в центр элемента, затем смещаем на (50, 100)
actions.moveToElement(targetElement)
       .moveByOffset(50, 100)
       .perform();
```

---
## Смещение от текущей позиции указателя

Перемещает мышь на указанное смещение от ее **текущего положения**.
```java
Actions actions = new Actions(driver);

int xOffset = 30;   // 30px вправо
int yOffset = -10;  // 10px вверх

actions.moveByOffset(xOffset, yOffset).perform();
```

> **Примечание:** Если мышь ранее не перемещалась, она находится в верхнем левом углу области просмотра. Позиция мыши не меняется при прокрутке страницы.

---
## Перетаскивание (Drag and Drop)

### dragAndDrop(source, target)
Перетаскивает исходный элемент (source) в целевой (target).
```java
WebElement sourceElement = driver.findElement(By.id("drag"));
WebElement targetElement = driver.findElement(By.id("drop"));

Actions actions = new Actions(driver);
actions
	.dragAndDrop(
		sourceElement, 
		targetElement
	)
	.perform();
```

> **Альтернатива:** Можно использовать комбинацию `clickAndHold()` + `moveToElement()` + `release()`.

---
## dragAndDropBy(source, xOffset, yOffset)

Перетаскивает элемент на указанное количество пикселей (смещение X и Y).

```java
WebElement sourceElement = driver.findElement(By.id("drag"));

Actions actions = new Actions(driver);
actions
	.dragAndDropBy(
		sourceElement, 
		150, 
		50
	)
	.perform(); // Смещаем на (150, 50)
```

# 3. Действия КЛАВИАТУРЫ (Keyboard Actions)

В Selenium WebDriver **действия клавиатуры автоматизируют нажатие и отпускание клавиш.** 

Это может быть комбинация из одного или нескольких событий, с помощью которых тестируется функциональность веб-приложения.

## Клавиши (Keys)
Клавиши представлены набором констант в виде Unicode-символов, представляющих специальные и непечатаемые клавиши на клавиатуре. Эти константы позволяют автоматизировать нажатия клавиш.

**Таблица 3-1. Базовые клавиши и их Unicode**

|Базовые клавиши|Unicode|Базовые клавиши|Unicode|
|---|---|---|---|
|NULL|\ue000|ENTER|\ue007|
|CANCEL|\ue001|SHIFT|\ue008|
|HELP|\ue002|CONTROL|\ue009|
|BACK_SPACE|\ue003|LEFT|\ue012|
|TAB|\ue004|UP|\ue013|
|CLEAR|\ue005|RIGHT|\ue014|
|RETURN|\ue006|DOWN|\ue015|

---
## Нажатие клавиши (keyDown)

Метод `keyDown()` имитирует **нажатие клавиши без её отпускания**. Используется, когда нужно вводить заглавные буквы (удерживая Shift) или выделять несколько элементов (удерживая Ctrl).
```java
Actions actions = new Actions(driver);

// Нажать и удерживать Shift
actions.keyDown(Keys.SHIFT).perform();
```

---
## Отпускание клавиши (keyUp)

Метод `keyUp()` **отпускает ранее нажатую клавишу**. Обычно используется в паре с `keyDown()` для имитации полного цикла "нажатие-отпускание".
```java
Actions actions = new Actions(driver);

// Нажать Shift, ввести текст, отпустить Shift
actions.keyDown(Keys.SHIFT)
       .sendKeys("hello") // Будет введено "HELLO"
       .keyUp(Keys.SHIFT)
       .perform();
```

---
## Ввод текста (sendKeys)

Метод `sendKeys()` используется **для ввода текста в веб-элементы** (поля ввода, поисковые строки, текстовые области). 
Его можно комбинировать с другими действиями в цепочке.

**Пример: ввод текста заглавными буквами**
```java
import org.openqa.selenium.By;
import org.openqa.selenium.Keys;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.firefox.FirefoxDriver;
import org.openqa.selenium.interactions.Actions;

public class KeyboardExample {
    public static void main(String[] args) {
        System.setProperty(
	        "webdriver.gecko.driver", "путь/к/geckodriver");
	        
        WebDriver driver = new FirefoxDriver();
        driver.get("https://google.com/");

        Actions actions = new Actions(driver);

        // Находим поле поиска, зажимаем Shift, вводим текст, отпускаем Shift
        actions.moveToElement(driver.findElement(By.name("q")))
               .keyDown(Keys.SHIFT)
               .sendKeys("java selenium book sujay")
               .keyUp(Keys.SHIFT)
               .perform();
    }
}
```

В этом примере текст `"JAVA SELENIUM BOOK SUJAY"` будет введен заглавными буквами, потому что клавиша Shift удерживается на протяжении всего ввода.

---
## Прокрутка (Scroll)

В Selenium WebDriver нет встроенной поддержки прокрутки, как у мыши или клавиатуры. Однако для прокрутки страницы вверх или вниз можно использовать:

1. **Класс `Actions`** (начиная с Selenium 4) — методы `scrollToElement()`, `scrollByAmount()`, `scrollFromOrigin()`.
    
2. **`JavascriptExecutor`** — универсальный способ, работающий во всех версиях.

---
### Прокрутка к элементу (scrollToElement)

Когда элемент находится за пределами области просмотра (viewport), используйте прокрутку, чтобы показать его.

**Способ 1: Через `Actions` (Selenium 4+)**
```java
WebElement element = 
	driver.findElement(By.linkText("Biomedicine"));

Actions actions = new Actions(driver);
actions
	.scrollToElement(element)
	.perform();
```

**Способ 2: Через `JavascriptExecutor` (универсальный)**
```java
WebElement element = 
	driver.findElement(By.linkText("Biomedicine"));

JavascriptExecutor js = (JavascriptExecutor) driver;
js.executeScript("arguments[0].scrollIntoView(true);", element);
// true — верх элемента совпадает с верхом viewport
// false — низ элемента совпадает с низом viewport
```

---
### Прокрутка на заданное количество пикселей (scrollByAmount)

Прокручивает страницу по горизонтали (deltaX) и вертикали (deltaY) на указанное количество пикселей.
```java
int deltaX = 50;  // 50 пикселей вправо
int deltaY = 100; // 100 пикселей вниз

new Actions(driver)
    .scrollByAmount(deltaX, deltaY)
    .perform();
```

Полезно для тестирования бесконечной прокрутки (infinite scroll) или ленивой загрузки (lazy loading).

---
### Прокрутка от элемента на заданное количество пикселей

Сначала определяется элемент, затем выполняется прокрутка от его центра на указанное смещение.
```java
import org.openqa.selenium.interactions.WheelInput;

// Находим элемент-ориентир
WebElement originElement = 
	driver.findElement(By.linkText("books"));

// Создаем точку отсчета: центр элемента со смещением (0, 100)
WheelInput.ScrollOrigin scrollOrigin = WheelInput.ScrollOrigin
    .fromElement(originElement, 0, 100);

// Прокручиваем от этой точки на 500 пикселей вниз
new Actions(driver)
    .scrollFromOrigin(scrollOrigin, 0, 500)
    .perform();
```

Здесь мы прокручиваем от точки, которая находится на 100 пикселей ниже центра элемента "books".

---
### Прокрутка от точки с заданным смещением

Прокрутка выполняется от определенной точки в области просмотра (viewport), заданной координатами X и Y от верхнего левого угла.
```java
import org.openqa.selenium.interactions.WheelInput;

// Точка отсчета: 100px вправо и 200px вниз от верхнего левого угла viewport
WheelInput.ScrollOrigin scrollOrigin = WheelInput.ScrollOrigin
    .fromViewport(100, 200);

// Прокручиваем от этой точки на 500 пикселей вниз
new Actions(driver)
    .scrollFromOrigin(scrollOrigin, 0, 500)
    .perform();
```
